# Part 2: McEasy Platform Mobile (MEP Mobile)

| Sub-category | Article | Key Content |
|---|---|---|
| **Get App** | How to get the mobile app | Available free on Google Play Store (Android) and App Store (iOS) |
| **Login & Logout** | Login with phone number | New option: add phone number + OTP verification; supports password change via WhatsApp |
| **Login & Logout** | Login with email | Enter email + password; Remember Me (1mo/2mo/1yr); CAPTCHA after 5 failed attempts; forgot password via 6-digit email OTP |
| **Login & Logout** | Logout | Profile → Keluar button |
| **Dasbor** | Access dashboard | Select Dasbor menu → set time period; 12 widgets: vehicle status, distance, top/lowest drivers, fuel, geofence vehicles |
| **Lisensi** | View vehicle licenses | Lisensi → Vehicle Tab; statuses: Berlaku / Akan Kedaluarsa (30 days) / Kedaluarsa / Tidak Teridentifikasi |
| **Lisensi** | View driver licenses | Lisensi → Driver Tab; same 4 statuses |
| **Penilaian Pengemudi** | View driver scores | Penilaian Pengemudi → set month → select driver → Lihat Papan Skor |
| **Pelacakan Sementara** | Create temporary tracking link | Pelacakan Sementara → Buat Baru → fill name/duration/info options → add vehicles → Save |
| **Pelacakan Sementara** | Share temporary tracking link | Select existing link → tap share icon in popup → choose sharing method |
| **Live View** | Read vehicle status | Statuses: Moving / Idle / Assisted / Parking / Offline; red GPS icon = no recent GPS data |
| **Live View** | View vehicle details | Live View → tap vehicle on map or search → swipe up on popup for full details |
| **Live View** | Filter vehicles | Filter by: vehicle status, vehicle category, driver status, vehicle group (single or multi-select) |
| **Live View** | Turn off engine | Vehicle detail → Matikan Mesin → confirm; engine locks until re-enabled via app |
| **Live View** | Turn on engine | Vehicle detail → Hidupkan Mesin → confirm |
| **Live View** | Contact driver | Vehicle detail → Hubungi → register phone number → connects to vehicle GSM |
| **Live View** | Share current location | Bagikan Lokasi → send one-time snapshot OR create temporary tracking link |
| **Live View** | View travel history | Set period/plate → filter by status → expand for details → animate trips; green = driving, gray = assisted |
| **Live View** | View vehicles in geolocation | Search icon → Geolokasi tab → select zone → see list of vehicles currently inside |
| **Live View** | Configure Live View display | Toggle: geolocation zones, vehicle clustering, traffic, Google location; Standard or Satellite map; vehicle/geo labels |
| **Notifikasi** | Enable notifications | Admin/Superadmin only; choose theme → select notifications → Activate; existing users can add themes |
| **Notifikasi** | View notification list | 3 tabs: driver behavior (speeding/harsh accel/sharp turns), general (GPS/temp/geofence), DMS TrackVision (with video playback) |
| **Profil** | View user information | Profile → Biodata Pengguna |
| **Pengukuran Lahan** | Create land measurement | Via Live View travel history OR Ukur Lahan menu → mark coordinate points → fill land name/work times → Save |
| **Pengukuran Lahan** | Edit land measurement | Ukur Lahan → select entry → Ubah → edit fields or adjust coordinates → Simpan |
| **Pengukuran Lahan** | Delete land measurement | Ukur Lahan → Hapus → check entries → Hapus Lahan → confirm |
| **Pelacakan Video** | View & download video history | Video Tracking → select vehicle → set date/time → browse folders → view/download (1-min segments) → Saved Videos |
| **Pelacakan Video** | View live video tracking | Home → Lacak Video → select running vehicle → view map + live stream; 1/2/4 camera grid; fullscreen mode |
| **Rangkuman** | Summary feature | Shows productivity rate, downtime, frequent notifications; daily/weekly/monthly view; browse past periods |
| **Laporan** | View reports | 6 report types: Travel History, Usage Summary, Travel Summary, Stop History, Fuel History, Temperature History; download as Excel/CSV/PDF |
| **2 Way Communication Using VoIP** | 2-way VoIP call to driver | Exclusive to TrackVision BSJ users; access via video tracking, vehicle details, or notifications; calls recorded in system |
